# Desafios de Vaga

Repositório para compilar desafios técnicos de processos seletivos para vagas de estágio e desenvolvedor júnior. Cada pasta contém a solução completa de um desafio diferente.

## Desafios

| Desafio | Empresa | Tecnologias | Descrição |
|---------|---------|-------------|-----------|
| [desafio-itau](./desafio-itau) | Itaú Unibanco | Java, Spring Boot | API REST para receber transações financeiras e retornar estatísticas (count, sum, avg, min, max) das transações dos últimos 60 segundos, sem uso de banco de dados — tudo em memória. |
| [desafio-magalu](./desafio-magalu) | Magalu | Java, Spring Boot, MySQL | API REST para agendar, consultar e cancelar envios de comunicações (Email, SMS, Push, WhatsApp) com persistência em banco de dados relacional. |
